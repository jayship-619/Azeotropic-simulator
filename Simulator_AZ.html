Key Equations Used in This Simulator
1 · Antoine Equation (Saturation Pressure)
log₁₀(Pᵢˢᵃᵗ) = Aᵢ − Bᵢ / (Cᵢ + T)
2 · NRTL Activity Coefficient (ln γᵢ)
ln γᵢ = [Σⱼ xⱼ·τⱼᵢ·Gⱼᵢ / Σₖ xₖ·Gₖᵢ] + Σⱼ [xⱼ·Gᵢⱼ / Σₖ xₖ·Gₖⱼ] · [τᵢⱼ − Σₖ xₖ·τₖⱼ·Gₖⱼ / Σₖ xₖ·Gₖⱼ]
where   Gᵢⱼ = exp(−αᵢⱼ · τᵢⱼ)
3 · Modified Raoult's Law (VLE)
yᵢ = xᵢ · γᵢ · Pᵢˢᵃᵗ / P  (raw)  →  normalise: yᵢ = yᵢ / Σyᵢ
4 · Bubble-Point Criterion
Σᵢ xᵢ · γᵢ · Pᵢˢᵃᵗ(T) = P  (solved iteratively for T)
5 · Relative Volatility
α₁₂ = (γ₁ · P₁ˢᵃᵗ) / (γ₂ · P₂ˢᵃᵗ)
6 · Overall & Component Mass Balance
F = D + B  ;  F·zᵢ = D·xDᵢ + B·xBᵢ
7 · Fenske Equation (Nmin)
Nmin = ln[(xD/(1−xD)) / (xB/(1−xB))] / ln(α₁₂)
8 · Underwood Equations (θ & Rmin)
Root θ:  Σᵢ [αᵢ·zᵢ / (αᵢ − θ)] = 1 − q
Rmin = Σᵢ [αᵢ·xDᵢ / (αᵢ − θ)] − 1
9 · Gilliland / Eduljee Correlation (Actual Stages N)
X = (R − Rmin) / (R + 1)
Y = 1 − exp[ ((1 + 54.4·X) / (11 + 117.2·X)) · ((X − 1) / X^0.5) ]
N = (Nmin + Y) / (1 − Y)
10 · Decanter LLE — Rachford–Rice (Phase-Split Fraction φ)
Σⱼ [zⱼ·(Kⱼ − 1) / (1 + φ·(Kⱼ − 1))] = 0  ;  Kⱼ = γⱼᵃq / γⱼᵒʳᵍ
xⱼᵃq = zⱼ / (1 + φ·(Kⱼ − 1))  ;  xⱼᵒʳᵍ = Kⱼ · xⱼᵃq
11 · Rectifying Operating Line (McCabe–Thiele)
y = [R / (R + 1)]·x + xD / (R + 1)
